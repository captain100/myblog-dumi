# vuepress 组织文档结构

> 使用 sidebar 配置文档分组

## 不能使用 index.md 命名

```javascript
// 在 .vuepress/config.js 的sidebar中不能设置index.md 文档无法找到文件

sidebar: [
  {
    title: 'typescript笔记',
    collapsable: false,
    children: [
      ['/startvuepress/index', 'index'], // 找不到文件
      ['/startvuepress/hahah  ', '使用vuepress'],
    ],
  },
];
```
